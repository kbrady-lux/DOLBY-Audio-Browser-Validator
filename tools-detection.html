What is ec-3 and ac-3?

Dolby Digital Plus, or Enhanced AC-3 (E-AC-3), is an advanced audio codec designed specifically for evolving media. It can be used in smartphone apps, online streaming services, PCs, tablets, smartphones and home theaters. Dolby Digital Plus defines high-fidelity audio even at low bit rates, employing exclusive compression algorithms to efficiently reduce the file size of digital audio programs. This allows for easier transmission and storage while delivering multichannel capability to bandwidth-constrained applications, mobile networks, and Internet-delivered entertainment.

Code Snippet

Use this javascript detection function test whether the browser supports Dolby Audio.

function DolbyTest() {
    // Check for Safari 8, as it will falsely report it can play back Dolby audio
    var is_safari = false;
    if ( navigator.userAgent.indexOf('Safari') != -1 && navigator.userAgent.indexOf('Version/8.0') != -1 ) { 
        is_safari = true; 
    }
    // Add a video element to the page
    var testEl = document.createElement( "video" );
    // Run the test
    if ( testEl.canPlayType && !is_safari ) { 
        return ( ( 'probably' === testEl.canPlayType('video/mp4;codecs="ec-3"') ) );  // Supports Dolby Audio
    } else {
        return false; // Dolby Audio Not Supported
    }
}

If it canPlayType, then display the Dolby version of the video. If not, then serve a stereo backup.

The media type tested for is MPEG-4 video using the Enhanced AC-3 codec: 'video/mp4;codecs="ec-3"'. Only browsers that report that they can 'probably' play this media pass the test. This is the strongest response possible. Some browsers return "maybe", indicating that they may be able to play the media, but in our tests, they could not. A response of 'maybe' does not pass the test.
